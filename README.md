# КупиПодариДай
API сервиса вишлистов КупиПодариДай
### ENV
Для запуска необходимо передать переменные окружения (.env)
```
PORT=3001
DB_HOST=localhost
DB_PORT=5432
DB_DATABASE=kupipodariday
DB_USERNAME=student
DB_PASSWORD=student
JWT_SECRET=some-jwt-secret
JWT_EXPIRES_IN=24h
SWAGGER_API_PATH=api
```
### Запуск
```git
npm run start
```
